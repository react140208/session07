## React Rendering

| Name | Full Name             | SEO | COST |
| ---- | --------------------- | --- | ---- |
| CSR  | Client Side Rendering | NO  | LOW  |
| SSR  | Server Side Rendering | YES | High |
| SSG  | Static Site Rendering | YES | Mid  |

### SSR

 - Server Express NodeJS
 - Usage: WebSite

### SSG
